**Full Stack & Security Engineer** — Brazil. I build production web platforms by day and tear things apart at the firmware / kernel level by night. Computer Science student, long-time Linux user, and contracted vulnerability researcher.

## What I actually do

- **Full Stack Engineer @ Grupo Multiluz** — React / TypeScript / Node.js / Angular / Bun, industrial SCADA (Modbus TCP/IP, Raspberry Pi), WhatsApp Cloud API platforms, PWAs, and RAG-based LLM integration for enterprise workflows.
- **Offensive Security Research & Responsible Disclosure** — contracted on multiple occasions to audit third-party software. Disclosed several major vulnerabilities under coordinated disclosure.

- **Freelance (SGS Canada, remote since 2021)** — C/C++ → Rust migrations and reliability automation.

## Featured projects

### Reverse Engineering & Low-Level

- **[ida-f32](https://github.com/GabriWar/ida-f32)** — IDA Pro processor module for AMD's F32 CP/MEC microcode ISA. Disassembles GPU command processor firmware.
- **[idaxport](https://github.com/GabriWar/idaxport)** — IDA Pro export plugin for AI-assisted analysis. Exports decompiled functions, types, xrefs, call graphs, prototypes, comments, vtables.
- **[gammakay61-linux](https://github.com/GabriWar/gammakay61-linux)** — Linux RGB controller for the Gammakay61 (SK61L) keyboard. USB HID protocol fully reverse-engineered from scratch.
- **[tozo-control](https://github.com/GabriWar/tozo-control)** — Native Linux GTK4 app to control TOZO earbuds over BLE (battery, EQ, game mode). No vendor Android app required.
- **[libmemgcc](https://github.com/GabriWar/libmemgcc)** — libmem cross-compiled for Windows MinGW g++.

### Tools & Fun Stuff

- **[kontrol](https://github.com/GabriWar/kontrol)** — Control your PC with hand gestures via webcam + MediaPipe.
- **[my98](https://github.com/GabriWar/my98)** — Web-based Windows 98 desktop recreation.
- **[vigilant](https://github.com/GabriWar/vigilant)** — Tiny Python script that watches webpages for changes.
