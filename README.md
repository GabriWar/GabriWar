# Hi, I'm Gabriel Duarte Guerra

**Full Stack & Security Engineer** — Brazil. I build production web platforms by day and tear things apart at the firmware / kernel level by night. Computer Science student, long-time Linux user, and contracted vulnerability researcher.

<h3> 🧠 &nbsp;What I actually do </h3>

- **Full stack engineering** — shipping production web platforms, real-time systems, and enterprise integrations across modern JS and backend stacks.
- **Offensive security research & responsible disclosure** — auditing software for vulnerabilities and reporting them through coordinated disclosure.
- **Reverse engineering at hardware & firmware level** — tearing apart binaries, proprietary device protocols, and low-level anti-tamper/DRM stacks.
- **Low-level & systems work** — C/C++, Rust, and Assembly on Linux, kernel internals, and embedded targets.
